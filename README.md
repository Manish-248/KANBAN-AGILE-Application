# Agile Kanban Task Management System

A Kanban-based Single Page Application (SPA) built with React.js to demonstrate Agile Project Management principles, workflow visualization, task tracking, and iterative software development.

---

## Project Overview

This project was developed as part of the Agile Project Management course to provide a practical implementation of Agile Kanban methodology through a modern React-based web application.

The application enables users to:

- Create and manage tasks
- Visualize workflow using a Kanban board
- Track task progress across workflow stages
- Assign priorities and responsibilities
- Search and filter tasks
- Monitor project statistics
- Manage work using Agile principles

The system follows a Single Page Application (SPA) architecture, ensuring real-time updates without page refreshes.
